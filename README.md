# tic-tac-toe-react-nest

Node v18.15.0
npm v9.5.0

## Set up

1. Clone the repo

```sh
$ git clone git@github.com:Alina-NZ/tic-tac-toe-react-nest.git
```

2. Change the current directory

```sh
$ cd tictactoe
```

3. Install all the dependencies

```sh
$ npm install
```

4. To start run:
`npm run dev`
